Getting Started


Instalation —

Node.js & NPM — To install Server • ily, you must first have Node.js installed along with NPM.

If you need to install Node.js and NPM on your machine, please refer to their documentation.
Node.js→ | NPM→


NPM Dependencies — "http-server" needs to be installed for Server • ily to run.

To install "http-server" open terminal, and type:   "npm install http-server -g"
Read http-server's Documentation ➜


Download Server • ily — To use Server • ily either clone the GitHub repository or download a .zip copy through GitHub.com

Using a Command Line Interface (CLI): To clone the github repository enter:   "git clone: https://github.com/apmcknight/template.git" into the terminal.

Downloading from GitHub.com: To download a ".zip" copy of the repository, visit GitHub.com/apmcknight/template, then click "Clone or Download"


Usage

Start Building — After you've installed dev dependencies, you can start building!

In terminal, change your directory   ($cd)   into the path you cloned the repo.
Then change directory into the cloned repository   (~pathname UserName$ cd template)

Once you're in the repository directory, enter   "npm start"   A http-server should start up, and you should see a message written in terminal:

Starting up http-server, serving ./app
Avaiable on: (YOUR IP ADDRESS)
