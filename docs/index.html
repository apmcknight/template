Getting Started:


DEPENDENCY REQUIREMENTS

To install and use our template, you must have Node installed locally on your machine. If you need to install Node.js and NPM on your machine, please refer to their documentation.   Node JS ‚Üí | NPM ‚Üí

After you install Node, run either "npm install", "npm i", or "yarn install" to pull dependencies and node_modules. Note: If you've installed the template from using our shell command, this should occur automatically.
